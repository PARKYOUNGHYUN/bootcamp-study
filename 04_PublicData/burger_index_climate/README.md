# 🥬 기후변화가 햄버거에 영향을 주는가 — 양상추 편

> 팀 프로젝트 **"기후변화가 햄버거에 영향을 주는가"** 중 **양상추 생산량과 기상 조건의 상관관계**를 분석

---

## 분석 개요

- 햄버거의 핵심 재료인 **양상추**의 생산량이 기후 변화에 따라 어떻게 달라지는지를 데이터로 분석
- 노지 상추와 시설 상추를 구분하여, 기온·강수·일사량 등 기상 지표와 생산량 간의 상관관계를 탐색

---

## 파일 구조

```
burger_index_climate/
├── eda.py          # 탐색적 데이터 분석 (생산량 추이 · 기상 분포 시각화)
├── main.py         # 기상 지표 × 생산량 상관관계 분석
├── README.md
└── data/
    ├── weather_data.csv    # 기상 관측 일별 데이터
    ├── yield_data.xlsx     # 엽채류 작물 생산량 데이터
    └── output/             # 실행 시 자동 생성 (차트 저장 폴더)
```

---

## 분석 내용

### `eda.py` — 탐색적 데이터 분석

| 분석 | 설명 |
|---|---|
| 상추 생산량 (톤) 추이 | 연도별 전체 생산량 변화 |
| 노지 / 시설 단수 추이 | 10a당 생산량 연도별 변화 |
| 재배방식별 단수 비교 | 노지 vs 시설 생산성 추이 비교 |
| 적온기온 빈도수 | 연도별 15~20°C 일수 분포 |
| 고온 빈도수 | 연도별 20.1°C 이상 일수 분포 |
| 집중호우 빈도수 | 연도별 80mm 이상 강수 발생 횟수 |
| 연간 온도 분포 (%) | 저온 / 적온 / 고온 비율 누적 막대 |

### `main.py` — 상관관계 분석

| 분석 | 설명 |
|---|---|
| 기본 기상 지표 × 생산량 | 연속강우·일사량과 생산량의 회귀 분석 |
| 월별 고온일수 × 생산량 | 월별로 20°C 이상 일수가 생산량에 미치는 영향 |
| 전월 극값 × 생산량 | 연간 고온일수·집중호우 빈도와 생산량 상관 |
| 재배기 상세 분석 | 최장 연속강우·무강수·강수변동성 등 지표 종합 분석 |
| 시나리오 분석 | 가뭄형 폭염 vs 장마형 폭염 시 평균 생산량 비교 |
| 최악 연도 분석 | 생산량 하위 3개년과 평년 기상 조건 비교 |
| 변동계수 분석 | 과거(2015~2019) vs 최근(2020~2024) 생산 안정성 비교 |

모든 분석은 **노지 상추**와 **시설 상추**를 각각 실행합니다.

## 주요 기상 지표 정의

| 지표 | 기준 |
|---|---|
| 적정온도 일수 (`optimum`) | 15.0 ~ 20.0°C |
| 온난 고온 일수 (`high_20_25`) | 20.1 ~ 24.9°C |
| 강 고온 일수 (`high_25_plus`) | ≥ 25.0°C |
| 집중호우 일수 (`heavy_rain`) | ≥ 80mm |
| 최장 연속 강우 (`max_streak`) | 재배기(3·4·5·9·10·11월) 기준 |
| 최장 무강수 (`max_dry_streak`) | 재배기 기준 |

## 실행 방법

```bash
# 탐색적 데이터 분석 (EDA)
python eda.py

# 상관관계 분석
python main.py
```

차트는 실행 시 `data/output/` 폴더에 자동 저장됩니다.
